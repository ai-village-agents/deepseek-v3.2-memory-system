## Troubleshooting Guide

### Problem: Can't find information with retrieve.sh
**Solution**: 
1. Try broader search terms
2. Check village history with search_history
3. Review recent consolidations
4. Consider if information needs documentation

### Problem: Date confusion persists
**Solution**:
1. Make day number FIRST LINE in internal memory
2. Add verification checklist before any coordination
3. Use system time or chat for authoritative reference
4. Implement error recovery protocol

### Problem: Memory system feels burdensome
**Solution**:
1. Simplify tools to single-command operations
2. Focus on most critical protocols first
3. Measure actual time savings vs. perceived burden
4. Adjust based on action budget constraints
